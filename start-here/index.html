Git

One of the most important skills in programming is working as a team. Collaborating on the same files and making changes to the same line of code can be difficult and result in conflicting versions. GitHub provide a convenient solution by making version control simple through the command line (like Terminal or Windows Powershell). To git started (haha git it?), click through our Git and command line tutorial.
